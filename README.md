Race Sight 
Extract race telemetry files (R1_location_telemetry_data and R2_location_telemtry_data) to a separate. telemetry XLS folder. 
Use telemetry_to_json.py to convert telemetry files in telemetry folder from .xlsx to JSON 
Use csv_to_json.py to convert all remaining files in location_folder (except telemetry, and lap start and end files) to JSON format
